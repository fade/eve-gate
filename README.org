#+TITLE: Eve-Gate: EVE Online ESI API Client
#+AUTHOR: Brian O'Reilly

A comprehensive EVE Online ESI API client library for Common Lisp.

* Welcome

Eve-Gate is a full-featured client for the EVE Online ESI API. It is designed to handle all the complexities of interacting with the API, allowing you to focus on building your application.

* Documentation

This project uses a comprehensive, multi-page documentation system written in Org-mode.

**For a quick start, and for all detailed documentation, please see the [[file:docs/index.org][Documentation Index]].**

The documentation covers:
-   Installation and a Quickstart Guide
-   In-depth guides on Authentication, Caching, and Concurrency
-   A complete API Reference
-   Architectural details

* Examples

For complete, runnable applications demonstrating key features, please see the [[file:examples/][examples/]] directory.
