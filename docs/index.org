* Eve-Gate Documentation

Welcome to the eve-gate documentation. Eve-gate is a comprehensive EVE Online
ESI (EVE Swagger Interface) API client library for Common Lisp.

** Important: Application Registration

#+begin_quote
Eve-gate is a library, not a hosted service. *You must register your own
EVE Developer application* at [[https://developers.eveonline.com/]] to use
authenticated endpoints. There is no shared Client ID - each developer
needs their own credentials. See [[file:authentication.org][Authentication]] for details.
#+end_quote

*** Installation

#+begin_src sh
cd ~/common-lisp/
git clone https://github.com/fade/eve-gate.git
#+end_src

*** Basic Usage

#+begin_src lisp
(ql:quickload :eve-gate)
(defvar *client* (eve-gate:make-eve-client))

;; Get server status
(eve-gate.api:get-status *client*)

;; Get character info
(eve-gate.api:get-characters-character-id *client* :character-id 95465499)
#+end_src

** Feature Overview

| Feature                  | Description                                    |
|--------------------------+------------------------------------------------|
| 195 API Functions        | Complete ESI endpoint coverage                 |
| OAuth 2.0                | Full EVE SSO authentication flow               |
| Multi-tier Caching       | Memory + file with ETag revalidation           |
| Rate Limiting            | Token bucket algorithm, per-endpoint limits    |
| Parallel Execution       | Bulk operations with concurrency control       |
| Type System              | EVE entity ID validation                       |
| Structured Logging       | JSON/text formatters with correlation IDs      |
| Health Monitoring        | System health checks and SLA tracking          |

** Package Overview

| Package               | Purpose                              |
|-----------------------+--------------------------------------|
| =eve-gate=              | Main client interface                |
| =eve-gate.api=          | 195 generated API functions          |
| =eve-gate.auth=         | OAuth 2.0 authentication             |
| =eve-gate.cache=        | Caching subsystem                    |
| =eve-gate.core=         | HTTP client and conditions           |
| =eve-gate.types=        | EVE entity types                     |
| =eve-gate.concurrent=   | Rate limiting and parallel ops       |
| =eve-gate.utils=        | Logging, config, monitoring          |
